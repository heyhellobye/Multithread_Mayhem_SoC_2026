# Week 2 – Synchronization and Classical Concurrency Problems

## Overview

This week focused on synchronization techniques and implementing classic multithreading problems using POSIX threads.

## Programs

| File | Description |
|------|-------------|
| `parallel_sum_fixed.c` | Parallel summation using multiple threads |
| `bank_chaos_fixed.c` | Race condition demonstration with synchronization |
| `bounded_buffer.c` | Producer–Consumer (Bounded Buffer) problem |
| `safe_list.c` | Thread-safe linked list implementation |
| `philosophers.c` | Deadlock-free Dining Philosophers solution |
| `philosophers_broken.c` | Naive Dining Philosophers implementation illustrating deadlock |

## Report

The detailed report is available in the `report` directory.
